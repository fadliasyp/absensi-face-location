Fitur Utama Sistem

Sistem dirancang untuk membantu proses absensi menjadi lebih aman, praktis, dan mudah dipantau.

🙂

Face Recognition

User dapat melakukan absensi dengan verifikasi wajah melalui kamera perangkat.

📍

Geolocation

Sistem mengecek apakah user berada dalam radius lokasi yang ditentukan admin.

👀

Liveness Detection

Verifikasi gerakan seperti kedip atau tengok untuk mengurangi risiko penggunaan foto.

📋

Panel Admin

Admin dapat mengelola user, lokasi, waktu masuk, dan data absensi.
